# Quill Writer Context Capsule

This document contains the authoritative context about the Quill Writer project.

## Project Overview

Quill Writer is a modern writing application built with Next.js, featuring a rich text editor powered by Novel.

## Architecture

- **Frontend**: Next.js 16 with React 19
- **Editor**: Novel editor library
- **Styling**: Tailwind CSS with custom themes
- **File System**: Uses File System Access API for local file operations

## Key Features

- Rich text editing with Novel
- File browser sidebar
- Theme customization
- Local file system integration
- Project-based organization
